sea と see

同音 /siː/。sea「海」（名詞）、see「見る・分かる」（動詞）。意味とスペルで判断する同音異義語。

音は同じ、意味とスペルで見分ける

sea と see は同じ発音 /siː/（同音異義語）。意味とスペルで決めます。

  • sea … /siː/ 名詞「海」。
  • see … /siː/ 動詞「見る・見える・分かる・会う」。see-saw-seen。
語発音品詞・中心の意味自作の例文
sea/siː/名詞：海We swam in the sea.（私たちは海で泳いだ）
see/siː/動詞：見る・分かるI can see the mountains from here.（ここから山が見える）
試験ポイント
覚え方：see は ee（二つの目）で「見る」。海は sea（ea）。動作「見る・分かる」なら see、水の「海」なら sea。

まとめ

  • sea と see は同音 /siː/
  • sea＝名詞「海」
  • see＝動詞「見る・分かる・会う」（ee＝二つの目）
