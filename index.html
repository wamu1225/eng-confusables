まぎらわしい英単語ノート

似て混同しやすい英単語を、引いてすぐ違いと使い分けがわかる逆引きの比較リファレンス。スペル・意味・品詞・発音が似ている250組の混同ペアを、比較表と例文でまとめています。

スペルが似ている語

  • affect と effect — 混同の超定番。affect は動詞「影響を与える」、effect は名詞「効果・結果」。品詞で割り切れます。
  • principal と principle — 同音。principal は「主要な／校長・元金」、principle は「原則・主義」。意味で分けます。
  • complement と compliment — 同音。真ん中の e と i で意味が変わる。complement は「補完」、compliment は「褒め言葉」。
  • stationary と stationery — 同音。-ary は形容詞「動かない」、-ery は名詞「文房具」。envelope の e で覚えます。
  • desert と dessert — s が1つか2つか。desert は「砂漠／見捨てる」、dessert は「デザート」。Sweet Stuff の ss。
  • altar と alter — -ar か -er か。altar は名詞「祭壇」、alter は動詞「変える」。alternative の仲間。
  • precede と proceed — 接頭辞で方向が逆。precede（pre＝前に）は「先行する」、proceed（pro＝前へ）は「進む」。
  • except と expect — 品詞が違う。except は前置詞「〜を除いて」、expect は動詞「予期する」。役割で見分けます。
  • ensure と assure と insure — 「保証」の3兄弟。事を確実に（ensure）／人を安心（assure）／保険（insure）で使い分け。
  • capital と capitol — ほぼ同音。capital は「首都・資本・大文字」、capitol は「議事堂」。ドームの O。
  • discreet と discrete — 同音。discreet は「慎重な・口が堅い」、discrete は「個別の・分離した」。e の位置で。
  • eminent と imminent — 頭が e か i か。eminent は「著名な」、imminent は「差し迫った」。Excellent/Immediate。
  • loose と lose — o の数と発音が違う。loose /luːs/「ゆるい」（形容詞）、lose /luːz/「失う・負ける」（動詞）。
  • quiet と quite — e と t の並びが逆。quiet /ˈkwaɪət/「静かな」（2音節）、quite /kwaɪt/「かなり・まったく」（1音節）。
  • than と then — a と e の一字違い。than /ðæn/「〜より」（比較）、then /ðen/「そのとき・それから」（時・順序）。
  • accept と except — 語頭 ac-/ex- が違う。accept「受け入れる」（動詞）、except「〜を除いて」（前置詞・接続詞）。
  • later と latter — t の数が違う。later /ˈleɪtər/「後で」（時間）、latter /ˈlætər/「後者」（前者に対する）。
  • angel と angle — e と l の並びが逆。angel /ˈeɪndʒəl/「天使」、angle /ˈæŋɡəl/「角度・角」。発音も意味も別。
  • through と thorough — つづりが紛らわしい。through /θruː/「〜を通って」（前置詞）、thorough /ˈθʌrə/「徹底的な」（形容詞）。
  • diary と dairy — ai と ir が入れ替わる。diary /ˈdaɪəri/「日記」、dairy /ˈdɛəri/「乳製品・酪農」。発音も別。
  • council と counsel — 同音 /ˈkaʊnsəl/。council「評議会・議会」（名詞）、counsel「助言（する）・弁護人」（名詞・動詞）。
  • lightning と lightening — e の有無で別語。lightning /ˈlaɪtnɪŋ/「稲妻」（名詞）、lightening /ˈlaɪtənɪŋ/「明るくする・軽くすること」（lighten の -ing）。
  • conscious と conscience — つづりが紛らわしい。conscious /ˈkɒnʃəs/「意識のある・気づいている」（形容詞）、conscience /ˈkɒnʃəns/「良心」（名詞）。
  • moral と morale — e の有無とアクセント。moral /ˈmɒrəl/「道徳的な・教訓」、morale /məˈræl/「士気・やる気」。
  • decent と descent — つづりが紛らわしい。decent /ˈdiːsənt/「きちんとした・まともな」（形容詞）、descent /dɪˈsɛnt/「下降・降下・家系」（名詞）。
  • born と borne — 同じ bear の過去分詞。born は「生まれる（be born）」の意味専用、borne はそれ以外（運ぶ・耐える・実らせる）の過去分詞。
  • canvas と canvass — s の数が違う。canvas「帆布・画布（キャンバス）」（名詞）、canvass「票や意見を求めて回る・世論調査する」（動詞）。
  • adverse と averse — adverse「不利な・逆の」（物事）、averse「（〜を）嫌って・気が進まない」（人・averse to）。
  • allusion と illusion — allusion「それとない言及・ほのめかし」、illusion「錯覚・幻想」。語頭 a-/i- が違う。
  • eligible と illegible — eligible「（〜の）資格がある・適格な」、illegible「判読できない・読めない（字）」。まったく別の語。
  • wander と wonder — wander /ˈwɑːndər/「あてもなく歩き回る」、wonder /ˈwʌndər/「不思議に思う・〜かしらと思う」。母音が違い、意味も別。
  • expand と expend — expand「大きくする・拡大する」、expend「（金・時間・労力を）費やす」。名詞形は expansion と expenditure。
  • perspective と prospective — perspective は名詞「観点・見方・遠近法」、prospective は形容詞「見込みの・将来の」。品詞から違う。
  • martial と marital — martial「軍事の・戦いの」（martial arts / martial law）、marital「結婚の・婚姻の」（marital status）。文字の並びが入れ替わっただけ。
  • access と excess — access「接近・利用（する権利）」、excess「超過・過剰」。アクセントの位置も違い、access は前、excess は後ろに置く。
  • statue と statute と stature — statue「彫像」、statute「制定法・法令」、stature「身長・（人物の）力量や名声」。1文字ずつ違う3語。
  • ingenious と ingenuous — ingenious「独創的な・巧妙な」、ingenuous「純真な・率直な」。i と u の1文字でほめ方の中身が変わる。
  • weary と wary — weary /ˈwɪri/「疲れ果てた・うんざりした」、wary /ˈweri/「用心深い」。e の有無で意味も母音も変わる。
  • bazaar と bizarre — bazaar「市場・バザー」（名詞）、bizarre「奇妙な・風変わりな」（形容詞）。発音はほぼ同じで、品詞と意味が違う。

意味が近く使い分けが難しい語

  • lie と lay — 自動詞 lie（横たわる）と他動詞 lay（〜を横たえる）。「目的語があるか」で見分け、活用の罠に注意。
  • rise と raise — 自動詞 rise（上がる）と他動詞 raise（〜を上げる）。目的語の有無と活用で見分けます。
  • borrow と lend — どちらも一時的なやり取りだが向きが逆。borrow は「借りる（受け取る側）」、lend は「貸す（渡す側）」。
  • beside と besides — s が付くだけで意味が変わる。beside は「〜のそばに」（位置）、besides は「〜に加えて」（追加）。
  • suitable と appropriate — どちらも「適切な」。suitable は「目的・用途に合う」、appropriate は「その場・社会的にふさわしい」。
  • profit と revenue — ビジネス頻出。revenue は「総収入・売上」、profit は「純利益（収入−経費）」。引き算の前か後か。
  • propose と suggest — どちらも「提案する」。propose は直接的・正式、suggest は控えめ。語法 suggest doing / suggest that に注意。
  • benefit と advantage — どちらも「利点」。benefit は「（直接的な）恩恵」、advantage は「（他と比べた）優位」。
  • say・tell・talk・speak — 「言う・話す」4兄弟。後ろにとる形（人を直接とるか、to が要るか）で見分けます。
  • fewer と less — 「より少ない」の使い分け。fewer は数えられる名詞（可算・複数）、less は数えられない名詞（不可算・量）に使う。
  • among と between — 「〜の間」の使い分け。between は2つ（または個々に見る複数）の間、among は3つ以上の不特定の集団の中。
  • bring と take — 移動の向きで使い分ける。bring は話し手（や話題の中心）の方へ「持ってくる」、take はそこから離す方へ「持っていく」。
  • remember と remind — 思い出す主体が違う。remember は自分が「思い出す・覚えている」、remind は人に「思い出させる」。
  • steal と rob — 目的語が違う。steal は「物」を盗む（steal something）、rob は「人・場所」から奪う（rob someone/a place）。
  • hear と listen — 意識の有無が違う。hear は自然に「聞こえる」、listen は意識して「聞く」（listen to）。
  • win と beat — 目的語が違う。win は「試合・賞」に勝つ（win a game）、beat は「相手」を負かす（beat someone）。
  • teach と learn — 向きが逆。teach は「（人に）教える」、learn は「（自分が）学ぶ・覚える」。
  • price と cost — 視点が違う。price は売り手がつける「売値」、cost は手に入れる/作るのに「かかる費用」。cost は動詞にもなる。
  • travel と trip と journey — 「旅」の3語。travel は移動全般（動詞・不可算）、trip は往復の（短い）旅行、journey は（長い）片道の道のり。
  • job と work — 数え方が違う。job は可算「（具体的な）仕事・職」、work は不可算「労働・仕事（一般）」。
  • make と do — 作り出すか、行うか。make は「（新しく）作り出す・生じさせる」、do は「（行為・仕事を）する・行う」。
  • damage と injure — 対象が違う。damage は「（物・もの）に損害を与える」、injure は「（人・体・動物）を傷つける・負傷させる」。
  • farther と further — farther は主に「（物理的な距離）より遠く」、further は「（程度・追加）さらに」＋距離にも使える。
  • reach と arrive — 自動詞か他動詞か。reach は他動詞で「（場所）に着く」（reach + 場所）、arrive は自動詞で at/in を伴う（arrive at/in）。
  • avoid と prevent — 主語の関わり方が違う。avoid は自分が「〜を避ける」、prevent は「（人・事が起きるの）を防ぐ・妨げる」。
  • big と large — big は大きさ・重要さ全般（口語的）、large は主にサイズ・量・数（ややフォーマル）。多くは置き換え可能だが相性がある。
  • salary と wage — 支払い方が違う。salary は主に「月給・年俸（固定給）」、wage は主に「時給・週給（働いた時間・量で払う賃金）」。
  • custom と habit — 主体が社会か個人か。custom は「（社会・集団の）慣習・風習」、habit は「（個人の）癖・習慣」。
  • shade と shadow — shade は「（光の当たらない）日陰・色合い」、shadow は「（物体がつくる輪郭のある）影」。
  • imply と infer — 向きが逆。imply は話し手が「ほのめかす・暗に示す」、infer は聞き手が「推測する・読み取る」。
  • refuse と reject — refuse は「（申し出・依頼）を断る」（refuse to do も）、reject は「（案・応募・品）を拒絶・却下する」。
  • notice と realize — notice は五感で「気づく・目に留める」、realize は頭で「（重要さ・事実を）はっきり理解する・悟る」。
  • home と house — house は「建物としての家」、home は「（暮らす場としての）家庭・我が家・故郷」。
  • receive と accept — receive は「（届いたものを）受け取る」（受動的）、accept は「（受け取ったものを）受け入れる・承諾する」（意志を伴う）。
  • allow と permit と let — 「許す」の3語。語法が違う。allow/permit は「人 to do」、let は「人 do（原形）」。permit はやや硬い。
  • earn と gain — earn は「（働き・努力で）稼ぐ・得る」、gain は「（量・程度を）増やして得る・獲得する」。
  • other と another — another＝an＋other で「もう一つの（単数・可算）」、other は「他の」（複数名詞や the とともに）。
  • amount と number — 数えられるかで使い分ける。amount は「（不可算の）量」、number は「（可算の）数」。amount of ~／number of ~。
  • much と many — 数えられるかで使い分ける。much は「（不可算に）多くの」、many は「（可算に）多くの」。
  • little と few — 数えられるかで使い分ける。little は「（不可算に）ほとんどない／少しの」、few は「（可算に）ほとんどない／少しの」。a の有無で意味が変わる。
  • hurt と injure と wound — 「傷つける」の3語。hurt は一般的（痛める・傷つける）、injure は事故などで負傷させる、wound は武器・刃物で傷を負わせる。
  • come と go — 向きが逆。come は話し手（や聞き手）の方へ「来る・行く」、go はそこから離れて「行く」。
  • hope と wish と expect — 「望む・思う」の3語。hope は実現可能なことを望む、wish は実現しにくい願い（仮定法）、expect は起こると予期する。
  • tall と high — tall は「（人・木・細長いものが）背が高い」、high は「（位置・程度が）高い／地面から高い所にある」。
  • rise と arise — どちらも自動詞。rise は「（物・数値などが）上がる・のぼる」、arise は「（問題・状況が）生じる・起こる」。
  • sit と seat — sit は自動詞「（自分が）座る」、seat は他動詞「（人を）座らせる」（ふつう be seated／seat oneself）。
  • emigrate と immigrate — 向きが逆。emigrate は「（自国から）出ていく」（emigrate from）、immigrate は「（他国へ）入ってくる」（immigrate to）。
  • biannual と biennial — 頻度が違う。biannual は「年に2回（半年ごと）」、biennial は「2年に1回」。
  • sit と set — 自動詞と他動詞。sit は「（自分が）座る」（自動詞）、set は「（物を）置く」（他動詞・set-set-set）。
  • look と see と watch — 「見る」の3語。look（at）は意識して視線を向ける、see は目に入る・見える、watch は動くものを継続して注視する。
  • during と while — どちらも「〜の間」。during は前置詞で後ろに名詞、while は接続詞で後ろに「主語＋動詞」の節が来る。
  • almost と most — almost は副詞「ほとんど・もう少しで」で名詞を直接修飾しない。「ほとんどの〜」は most 〜 か almost all 〜。
  • client と customer — customer は商品やサービスを買う客、client は専門的サービスを継続的に受ける依頼人。日本語ではどちらも「顧客」。
  • assume と presume — どちらも「思い込む・推定する」。assume は根拠なしに前提として置く、presume はある程度の根拠から推定する。
  • especially と specially — especially「とりわけ・特に（程度が際立つ）」、specially「特別に（ある目的のために）」。
  • each と every — each は2つ以上を1つずつ個別に見る、every は3つ以上を全体としてまとめて見る。どちらも動詞は単数扱い。
  • still と yet と already — still「まだ続いている」、yet「まだ〜ない・もう〜したか」、already「もう・すでに」。位置と使う文の種類が違う。
  • used to と be used to — used to do「以前は〜した（今は違う）」、be used to doing「〜に慣れている」。後ろが原形か -ing かで意味が変わる。

品詞・派生でまぎらわしい語

  • economic と economical — 同じ語根で -ic か -ical か。economic は「経済の」、economical は「節約的・お得な」。
  • historic と historical — historic は「歴史に残るほど重要な」、historical は「歴史上の・過去の（重要かは問わない）」。
  • sensible と sensitive — sensible は「分別がある・賢明な」、sensitive は「敏感な・神経質な」。sense の使い方が逆。
  • respectable・respectful・respective — respect の3兄弟。立派な（respectable）／敬意を払う（respectful）／それぞれの（respective）。
  • industrial と industrious — industrial は「工業の・産業の」（もの）、industrious は「勤勉な」（人）。-al か -ous か。
  • considerable と considerate — considerable は「かなりの（量・程度）」、considerate は「思いやりのある（人）」。-able か -ate か。
  • effective と efficient — effective は「効果的（結果を出す）」、efficient は「効率的（無駄がない）」。結果か効率か。
  • personal と personnel — personal は形容詞「個人の・私的な」、personnel は名詞「職員・人員・人事」。アクセント位置も違う。
  • complain と complaint — complain は動詞「苦情を言う」、complaint は名詞「苦情」。-t が付くと名詞、で見分けます。
  • imaginative・imaginary・imaginable — imagine の3兄弟。想像力豊か（-ative）／架空の（-ary）／考えうる（-able）。接尾辞で区別。
  • literal・literary・literate — literal は「文字どおりの」、literary は「文学の」、literate は「読み書きできる」。
  • institution と instruction — スペルが似た別語。institution は「機関・制度」、instruction は「指示・説明・指導」。
  • permission と admission — permission は「（行為の）許可」、admission は「入場・入学（の許可）／入場料／（罪の）認め」。
  • agent と agency — agent は「代理人・行為者（人）」、agency は「代理店・機関（組織）」。人か組織か。
  • related と relevant — related は「（単に）関連した」、relevant は「（その話題に）関連があり重要・適切な」。
  • advice と advise — advice は名詞「助言」、advise は動詞「助言する」。語尾 c か s かで品詞が変わる。
  • adapt と adopt — a か o か。adapt は「適応させる・改造する」、adopt は「採用する・養子にする」。
  • classic と classical — 派生でまぎらわしい。classic「一流の・典型的な・名作」、classical「古典の・クラシックの（音楽・様式）」。
  • childish と childlike — 同じ child でも評価が逆。childish「子どもっぽい（否定的・幼稚）」、childlike「子どものような（肯定的・純真）」。
  • successful と successive — 同じ success 由来でも別語。successful「成功した」、successive「連続する・引き続く」。
  • alternate と alternative — alternate「交互の・一つおきの／代わりを務める」、alternative「代わりの・別の選択肢（の）」。
  • device と devise — 名詞と動詞。device /dɪˈvaɪs/「装置・工夫」（名詞）、devise /dɪˈvaɪz/「考案する」（動詞）。語尾 /s/ と /z/。
  • breath と breathe — 名詞と動詞。breath /brɛθ/「息」（名詞）、breathe /briːð/「呼吸する」（動詞）。母音と語尾の音が違う。
  • cloth と clothe — 名詞と動詞。cloth /klɔːθ/「布・生地」（名詞）、clothe /kloʊð/「（人に）服を着せる」（動詞）。発音も違う。
  • belief と believe — 名詞と動詞。belief /bɪˈliːf/「信念・信じること」（名詞）、believe /bɪˈliːv/「信じる」（動詞）。語尾 /f/ と /v/。
  • relief と relieve — 名詞と動詞。relief /rɪˈliːf/「安心・緩和」（名詞）、relieve /rɪˈliːv/「（苦痛などを）和らげる」（動詞）。語尾 /f/ と /v/。
  • envelope と envelop — 名詞と動詞。envelope /ˈɛnvəloʊp/「封筒」（名詞）、envelop /ɪnˈvɛləp/「包み込む」（動詞）。e の有無とアクセントが違う。
  • proof と prove — 名詞と動詞。proof /pruːf/「証拠・証明」（名詞）、prove /pruːv/「証明する」（動詞）。語尾 /f/ と /v/。
  • success と succeed — 名詞と動詞。success /səkˈsɛs/「成功」（名詞）、succeed /səkˈsiːd/「成功する・後を継ぐ」（動詞）。
  • confident と confidant — confident /ˈkɒnfɪdənt/「自信のある」（形容詞）、confidant /ˈkɒnfɪdænt/「腹心の友・秘密を打ち明ける相手」（名詞）。
  • momentary と momentous — 同じ moment 由来でも別語。momentary /ˈmoʊməntɛri/「一瞬の・つかの間の」、momentous /moʊˈmɛntəs/「重大な」。
  • luxury と luxurious — 品詞が違う。luxury /ˈlʌkʃəri/「ぜいたく・高級品」（名詞）、luxurious /lʌɡˈʒʊəriəs/「豪華な・ぜいたくな」（形容詞）。
  • healthy と healthful — healthy「健康な・健康的な」（人・状態）、healthful「健康に良い」（食べ物・環境など、健康を促すもの）。※現代英語では healthy が両方に使われる傾向。
  • comprehensible と comprehensive — 同じ comprehend 由来でも別語。comprehensible「理解できる・わかりやすい」、comprehensive「包括的な・網羅的な」。
  • credible と credulous — 同じ cred（信じる）由来でも別語。credible「信用できる・信じるに足る」、credulous「（人が）すぐ信じる・だまされやすい」。
  • practical と practicable — practical「実用的な・現実的な」（役に立つ・地に足のついた）、practicable「実行可能な」（実際にやれる）。
  • desirable と desirous — desirable「望ましい・好ましい」（〜される側）、desirous「（〜を）望んでいる」（望む側・desirous of）。
  • regretful と regrettable — regretful「（人が）後悔している・残念に思う」、regrettable「（事が）残念な・遺憾な」。主語が人か事かで使い分ける。
  • official と officious — official「公式の・職員」、officious「お節介な・でしゃばりな」。同じ office 由来だが意味は大きく違う。
  • flammable と inflammable — 要注意：flammable と inflammable は同じ意味「燃えやすい」。in- は否定でなく強めの接頭辞。「燃えない」は non-flammable。
  • famous と infamous — famous「有名な（よい意味も含む）」、infamous「悪名高い（悪い評判で有名）」。in- がここでは否定的な色を加える。
  • memorable と memorial — memorable「記憶に残る・忘れられない」（形容詞）、memorial「記念の・追悼の（形容詞）／記念碑（名詞）」。
  • forgetful と forgettable — forgetful「（人が）忘れっぽい」、forgettable「（物・事が）忘れられやすい・印象に残らない」。
  • intelligent と intelligible — intelligent「頭がいい・知的な」（人・生物）、intelligible「（言葉・説明が）理解できる・分かりやすい」（物事）。
  • sociable と social — sociable「社交的な・人付き合いのよい」（人の性格）、social「社会の・社交（に関する）」（社会・集まり全般）。
  • continual と continuous — continual は「繰り返し起こる・断続的な」、continuous は「途切れずに続く」。
  • human と humane — human「人間の・人間」、humane「人道的な・思いやりのある」。末尾 e とアクセントで意味が変わる。
  • confusing と confused — 感情動詞の -ing 形と -ed 形。confusing は「（物事が）混乱させる」、confused は「（人が）混乱している」。
  • comparable と comparative — comparable「比較できる・同等の」、comparative「比較による・相対的な」（文法では比較級）。compare の派生2語。
  • complementary と complimentary — complementary「補い合う・補完的な」、complimentary「賛辞の／無料の」。e と i の1文字で意味が変わる。
  • hard と hardly — hard は副詞で「熱心に・激しく」、hardly は「ほとんど〜ない」。-ly をつけると意味が強まるどころか否定に変わる。
  • late と lately — late は形容詞・副詞「遅い・遅く」、lately は副詞「最近」。-ly で時刻の話から期間の話に変わる。
  • cook と cooker — cook は「料理する」／「料理人」、cooker は「調理器具（コンロ・レンジ）」。-er が人を表さない例。
  • invaluable と valueless — invaluable は「（値がつけられないほど）非常に貴重な」、valueless は「無価値な」。in- が否定に見えるのが罠。
