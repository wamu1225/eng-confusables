まぎらわしい英単語ノート

似て混同しやすい英単語を、引いてすぐ違いと使い分けがわかる逆引きの比較リファレンス。スペル・意味・品詞・発音が似ている51組の混同ペアを、比較表と例文でまとめています。

スペルが似ている語

  • affect と effect — 混同の超定番。affect は動詞「影響を与える」、effect は名詞「効果・結果」。品詞で割り切れます。
  • principal と principle — 同音。principal は「主要な／校長・元金」、principle は「原則・主義」。意味で分けます。
  • complement と compliment — 同音。真ん中の e と i で意味が変わる。complement は「補完」、compliment は「褒め言葉」。
  • stationary と stationery — 同音。-ary は形容詞「動かない」、-ery は名詞「文房具」。envelope の e で覚えます。
  • desert と dessert — s が1つか2つか。desert は「砂漠／見捨てる」、dessert は「デザート」。Sweet Stuff の ss。
  • altar と alter — -ar か -er か。altar は名詞「祭壇」、alter は動詞「変える」。alternative の仲間。
  • precede と proceed — 接頭辞で方向が逆。precede（pre＝前に）は「先行する」、proceed（pro＝前へ）は「進む」。
  • except と expect — 品詞が違う。except は前置詞「〜を除いて」、expect は動詞「予期する」。役割で見分けます。
  • ensure と assure と insure — 「保証」の3兄弟。事を確実に（ensure）／人を安心（assure）／保険（insure）で使い分け。
  • capital と capitol — ほぼ同音。capital は「首都・資本・大文字」、capitol は「議事堂」。ドームの O。
  • discreet と discrete — 同音。discreet は「慎重な・口が堅い」、discrete は「個別の・分離した」。e の位置で。
  • eminent と imminent — 頭が e か i か。eminent は「著名な」、imminent は「差し迫った」。Excellent/Immediate。

意味が近く使い分けが難しい語

  • lie と lay — 自動詞 lie（横たわる）と他動詞 lay（〜を横たえる）。「目的語があるか」で見分け、活用の罠に注意。
  • rise と raise — 自動詞 rise（上がる）と他動詞 raise（〜を上げる）。目的語の有無と活用で見分けます。
  • borrow と lend — どちらも一時的なやり取りだが向きが逆。borrow は「借りる（受け取る側）」、lend は「貸す（渡す側）」。
  • beside と besides — s が付くだけで意味が変わる。beside は「〜のそばに」（位置）、besides は「〜に加えて」（追加）。
  • suitable と appropriate — どちらも「適切な」。suitable は「目的・用途に合う」、appropriate は「その場・社会的にふさわしい」。
  • profit と revenue — ビジネス頻出。revenue は「総収入・売上」、profit は「純利益（収入−経費）」。引き算の前か後か。
  • propose と suggest — どちらも「提案する」。propose は直接的・正式、suggest は控えめ。語法 suggest doing / suggest that に注意。
  • benefit と advantage — どちらも「利点」。benefit は「（直接的な）恩恵」、advantage は「（他と比べた）優位」。
  • say・tell・talk・speak — 「言う・話す」4兄弟。後ろにとる形（人を直接とるか、to が要るか）で見分けます。

品詞・派生でまぎらわしい語

  • economic と economical — 同じ語根で -ic か -ical か。economic は「経済の」、economical は「節約的・お得な」。
  • historic と historical — historic は「歴史に残るほど重要な」、historical は「歴史上の・過去の（重要かは問わない）」。
  • sensible と sensitive — sensible は「分別がある・賢明な」、sensitive は「敏感な・神経質な」。sense の使い方が逆。
  • respectable・respectful・respective — respect の3兄弟。立派な（respectable）／敬意を払う（respectful）／それぞれの（respective）。
  • industrial と industrious — industrial は「工業の・産業の」（もの）、industrious は「勤勉な」（人）。-al か -ous か。
  • considerable と considerate — considerable は「かなりの（量・程度）」、considerate は「思いやりのある（人）」。-able か -ate か。
  • effective と efficient — effective は「効果的（結果を出す）」、efficient は「効率的（無駄がない）」。結果か効率か。
  • personal と personnel — personal は形容詞「個人の・私的な」、personnel は名詞「職員・人員・人事」。アクセント位置も違う。
  • complain と complaint — complain は動詞「苦情を言う」、complaint は名詞「苦情」。-t が付くと名詞、で見分けます。
  • imaginative・imaginary・imaginable — imagine の3兄弟。想像力豊か（-ative）／架空の（-ary）／考えうる（-able）。接尾辞で区別。
  • literal・literary・literate — literal は「文字どおりの」、literary は「文学の」、literate は「読み書きできる」。
  • institution と instruction — スペルが似た別語。institution は「機関・制度」、instruction は「指示・説明・指導」。
  • permission と admission — permission は「（行為の）許可」、admission は「入場・入学（の許可）／入場料／（罪の）認め」。
  • agent と agency — agent は「代理人・行為者（人）」、agency は「代理店・機関（組織）」。人か組織か。
  • related と relevant — related は「（単に）関連した」、relevant は「（その話題に）関連があり重要・適切な」。
  • advice と advise — advice は名詞「助言」、advise は動詞「助言する」。語尾 c か s かで品詞が変わる。
  • adapt と adopt — a か o か。adapt は「適応させる・改造する」、adopt は「採用する・養子にする」。

発音が似ている語

  • breed と bleed — /r/ と /l/ の違い。breed /briːd/「繁殖させる・品種」、bleed /bliːd/「出血する」。
  • grow と glow — /r/ と /l/ の違い。grow /ɡroʊ/「成長する」、glow /ɡloʊ/「輝く・赤熱する」。
  • flow と flaw — 母音 /oʊ/ と /ɔː/ の違い。flow /floʊ/「流れる」、flaw /flɔː/「欠点・きず」。
  • break と brake — 同音 /breɪk/。break「壊す・休憩」、brake「ブレーキ・制動する」。文脈とスペルで判断。
  • bear と bare — 同音 /beə(r)/。bear「クマ／耐える・運ぶ」、bare「裸の・むき出しの」。文脈で判断。
  • ant と aunt — ant「アリ」、aunt「おば」。米発音では同音 /ænt/、英発音では aunt /ɑːnt/ と異なる。
  • brush と blush — /r/ と /l/ の違い。brush /brʌʃ/「ブラシ・磨く」、blush /blʌʃ/「赤面する」。
  • room と loom — /r/ と /l/ の違い。room /ruːm/「部屋」、loom /luːm/「織機／ぬっと現れる・迫る」。
  • crash・crush・clash — 母音 /æ/ /ʌ/ と /r/ /l/ の複合。crash 衝突・墜落／crush 押しつぶす・夢中／clash 対立・衝突。
  • low と law — 母音 /oʊ/ と /ɔː/ の違い。low /loʊ/「低い」、law /lɔː/「法・法律」。
  • raw と row — 母音 /ɔː/ と /oʊ/ の違い。raw /rɔː/「生の・未加工の」、row /roʊ/「列／（舟を）漕ぐ」。
  • cell と sell — 同音 /sel/。cell「細胞・独房・電池・携帯」、sell「売る」。文脈とスペルで判断。
  • claw と crow — /l/ /r/ と母音 /ɔː/ /oʊ/ の複合。claw /klɔː/「（動物の）爪」、crow /kroʊ/「カラス」。
