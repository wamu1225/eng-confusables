accept と except

語頭 ac-/ex- が違う。accept「受け入れる」（動詞）、except「〜を除いて」（前置詞・接続詞）。

品詞と語頭で見分ける

語頭が ac-（accept）と ex-（except） で、意味も品詞も違います。

  • accept … /əkˈsept/ 動詞「受け入れる・受け取る・認める」。
  • except … /ɪkˈsept/ 前置詞・接続詞「〜を除いて・〜以外は」。
語発音品詞・中心の意味自作の例文
accept/əkˈsept/動詞：受け入れるShe accepted the job offer.（彼女は内定を受けた）
except/ɪkˈsept/前置詞：〜を除いてEveryone came except Tom.（トム以外はみんな来た）
試験ポイント
覚え方：ex- は「外へ（out）」＝そこから外す「除いて」。accept は ac-（〜へ）で「受け入れる」動詞。文で動詞が必要なら accept、「〜以外」なら except。

まとめ

  • accept /əkˈsept/＝動詞「受け入れる・受け取る」
  • except /ɪkˈsept/＝前置詞・接続詞「〜を除いて」
  • ex-（外へ）＝除く、ac-（〜へ）＝受け入れる
